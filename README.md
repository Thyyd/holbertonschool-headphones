# holbertonschool-headphones

Dans ce projet, nous allons mettre en pratique ce que nous avons appris dans les dernieres projets afin de créer un site responsif de A à Z à partir d'un wireframe donné.

## Objectifs

- Créer un site à partir d'un wireframe
- Adapter ce site pour le rendre responsif pour smartphones, tablettes et laptop

## Technologies

- Langage HTML
- Langage CSS

## Fonctionnalités

- Header avec Logo et plusieurs liens
- Main contenant :
  - Partie Principale, que l'on voit directement quand on arrive sur le site
  - Partie "What do we do...", montrant les principales choses que proposent le site
  - Partie Résultats, montrant les résultats qu'ils ont obtenus
  - Partie Contact Us, contenant un bouton "Call to Action" et 3 champs label
- Footer avec informations de contact
